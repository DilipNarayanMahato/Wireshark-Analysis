# Network Traffic Analysis using Wireshark

## Description
This project demonstrates network traffic analysis using Wireshark. The goal is to capture and examine network packets to understand communication patterns, identify protocols, and detect potential anomalies.

## Tools Used
- Wireshark
- Linux (optional for packet capture)
- TCP/IP Network Fundamentals

## What I Did
- Captured live network packets in a lab environment.
- Analyzed protocols such as HTTP, TCP, UDP, and DNS.
- Observed traffic patterns and identified unusual or suspicious activity.
- Documented findings in a structured report.

## Learning Outcomes
- Practical experience in network packet analysis.
- Understanding of common network protocols and traffic flows.
- Basics of network troubleshooting and security monitoring.
